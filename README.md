## v2 — live-data changes vs v1

1. **14 US FDA recalls surfaced** in the 5-year window — 1 × Class I (Staska Pharmaceuticals, glass particulates in Ascorbic Acid Injection 500 mg/mL, 2024-09-30), 12 × Class II (compounding-pharmacy sterility cluster on IV ascorbic acid), 1 × Class III (Teva/Lupin PEG bowel prep pouch leakage). A new emerging-signal row — "Compounded-IV sterility / particulate contamination" — was added to formal signal evaluation, raising open workflows from 4 to 5.
2. **True FAERS 2×2 RRR computed** (vs v1's crude proxy). Haemolysis RRR 3.48 and Nephrolithiasis RRR 2.33 both cross the formal-signal threshold (2.0); Phlebitis 1.93, Dermatitis contact 1.93, AKI 1.70, Haemolytic anaemia 1.60 sit on the watchlist.
3. **FAERS expansion** — 402,567 total cases, 32.9 % serious, 7.9 % fatal. Route breakdown: Oral 222,358 · SC 108,113 · IV 71,566 · Topical 18,574. Yearly trend 2021-2025 declining 28.3k → 21.9k.
4. **Four ingredient-identity corrections** from PubChem live: calcium ascorbate CAS 5743-27-1 · UNII J96U0ZD4Y6; ascorbyl palmitate UNII QN83US2B0N; tetrahexyldecyl ascorbate UNII 47143LT58A. RxCUI 1151 added.
5. **MHRA / EMA PRAC / Health Canada drug-PV channels confirmed genuinely empty** (not a tooling artifact) via live scrape.

## Data provenance

### Live / real calls (v2)

- PubMed E-utilities (no key)
- openFDA `drug/event`, `drug/label`, `drug/enforcement` (no key)
- PubChem PUG REST + RxNav (no key)
- MHRA / gov.uk (HTML, server-rendered, parseable)
- EMA — substance search + 10 recent PRAC minutes / agendas / highlights (HTML)
- Health Canada `recalls-rappels.canada.ca` (HTML)

### Blocked or still simulated

- **Credentialed only:** Embase (Elsevier), Scopus (Elsevier), Web of Science (Clarivate), Cochrane API, LILACS (no public JSON), EudraVigilance line listings (MAH / DAP), VigiBase / VigiLyze (WHO-UMC), MedDRA MSSO.
- **Bot-mitigation / SPA-gated:** TGA Safety Alerts + MSU + DAEN — all endpoints returned timeout / 403 / SPA gate. Production path is a headless-browser scrape or the TGA DAEN bulk CSV export.
- **Feed retired:** FDA Drug Safety Communications RSS — all three candidate URLs returned HTTP 404. DSC absence for ascorbate is inferred via label + enforcement pull, not directly confirmed via an RSS scan.
