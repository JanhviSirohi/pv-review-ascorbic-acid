## Data caveats

- **Real API calls:** PubMed via E-utilities (Phase 6 literature search), openFDA FAERS (Phase 7 PV database).
- **Simulated shells** (flagged `simulated: true` in source outputs): Embase, Cochrane, Scopus, Web of Science, LILACS, EudraVigilance, VigiBase, TGA DAEN.
- **Regulatory communications scan** (Phase 8) returned empty — live WebFetch of regulator listing pages is required at run time; fabrication was explicitly avoided.
- Fields marked `configured by user` (products / brands, internal product code, countries marketed) are populated from the company product master data in production.

This is a demonstration of pipeline structure and output fidelity, not a production regulatory deliverable.
